## üì¶ Component Categories

The library is organized into six main categories:

### 1. **Core Components** (`core/`)
Essential UI building blocks:
- Buttons, Forms, Layout, Typography
- Media, Data Display, Feedback
- Navigation, Overlays, Modals

### 2. **Admin Components** (`admin/`)
Admin panel and dashboard components:
- Dashboards, KPI cards, Charts
- CRUD tables, Data management
- User roles and permissions

### 3. **B2B Components** (`b2b/`)
Enterprise workspace features:
- Workspace and team management
- Projects, clients, invoices
- Kanban boards, workflows
- Approval systems

### 4. **E-commerce Components** (`ecommerce/`)
Online store components:
- Product listings and details
- Shopping cart and checkout
- Order management
- Customer accounts

### 5. **Marketing Components** (`marketing/`)
Marketing and content sections:
- Hero sections, Features
- Testimonials, Pricing
- Blog layouts, CTAs

### 6. **Utility Components** (`utility/`)
Cross-cutting utilities:
- Auth layouts, Error boundaries
- Search bars, Filters

See [ARCHITECTURE.md](ARCHITECTURE.md) and [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md) for complete component lists.
